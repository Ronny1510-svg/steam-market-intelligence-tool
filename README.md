# steam-market-intelligence-tool.
Solución de automatización avanzada y Web Scraping para la extracción, limpieza y jerarquización de ofertas de mercado. Utiliza Selenium y Pandas para transformar datos web dinámicos en reportes estratégicos de Excel con formato profesional.
**Transforma datos no estructurados en decisiones de compra estratégicas.**

Este software es una solución de automatización avanzada que convierte la gran base de datos de Steam en un documento estructurado y gerarquizado, listo para el análisis de negocios.

---

## ¿Para quién es esto?
Este proyecto fue diseñado pensando en perfiles empresariales que requieren agilidad y precisión:
* **Analistas de Mercado:** Para monitorear fluctuaciones de precios en la industria del entretenimiento digital.
* **Inversionistas y Smart Buyers:** Para optimizar presupuestos identificando instantáneamente las oportunidades con mayor ahorro.
* **Data Analysts:** Para obtener datasets limpios (Excel) sin el ruido del contenido web no estructurado.

---

## Valor Agregado.
En el mercado actual, la información oportuna es una ventaja competitiva. Este scraper ofrece:
1. **Eficiencia Operativa:** Automatiza la recolección de datos, reduciendo procesos manuales de varias horas a escasos **15 segundos**.
2. **Integridad de Datos:** Captura precios finales exactos, eliminando el error humano en la transcripción de ofertas complejas.
3. **Reportes Listos para la Gerencia:** Genera archivos `.xlsx` con formato profesional

https://github.com/user-attachments/assets/6929f529-4373-4a98-8ec2-dc49361a9719

 listos para integrar en dashboards de Power BI o presentaciones ejecutivas.

## Ingeniería detrás del Producto
Para garantizar la entrega de datos incluso ante bloqueos de seguridad, el software implementa:
* **Navegación Robusta:** Motor de **Selenium** para el renderizado completo de contenido dinámico (JavaScript).
* **Inteligencia de Datos:** Pipeline de limpieza con **Pandas** que jerarquiza la información por relevancia y magnitud de descuento.
* **UX en Reportes:** Integración con **Openpyxl** para entregar un producto final con legibilidad premium.
